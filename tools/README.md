# Quelques outils

## `crew.py`

Utilisation :

```shell
python crew.py
```

Génère les sept membres d'équipages (association styles & rôles).

## `threat.py`

Utilisation :

```shell
python threat.py [nb]
```

Génère une ou plusieurs menaces en utilisant les tables prévues à cet effet.
Si "nb" n'est pas précisé, une seule menace est tirée au hasard.
